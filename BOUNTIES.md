# 🎯 Régisseur IA - Bounties & Most Wanted Plugins

Bienvenue sur le Leaderboard des contributions ! Le but de ce projet est de devenir le standard open source de la régie improvisée et du spectacle vivant assisté par IA.

Aidez-nous à construire l'écosystème en développant ces plugins très demandés. Gagnez des points, grimpez dans le Leaderboard et devenez un contributeur majeur.

## 🏆 Le Barème (Most Wanted)

| Plugin | Description | Points | Statut |
|---|---|---|---|
| **QLab / OSC** | Intégration du standard de l'industrie (QLab) via requêtes OSC. | 100 Pts | 🟢 Ouvert |
| **Apple Music** | Alternative à Spotify, très demandée par les utilisateurs macOS. | 80 Pts | 🟢 Ouvert |
| **Deezer / YT Music** | Autres plateformes de streaming pour éviter le monopole. | 50 Pts | 🟢 Ouvert |
| **DMX / Art-Net** | Le Graal : synchroniser les lumières de la salle avec les actions audio. | 150 Pts | 🟢 Ouvert |
| **OBS Studio** | Contrôler le stream vidéo ou lancer des overlays via WebSockets. | 70 Pts | 🟢 Ouvert |

## 🛠️ Comment participer ?
1. Jetez un œil au fichier `ARCHITECTURE.md` pour comprendre comment fonctionne le système de *Plugins*.
2. Forkez le repo et créez votre fichier dans le dossier `plugins/` (ex: `qlab_plugin.py`).
3. Faites une Pull Request avec le tag `[BOUNTY]`.
